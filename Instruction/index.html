网站介绍

主要技术

HTML

HTML称为超文本标记语言，是一种标识性的语言。它包括一系列标签．通过这些标签可以将网络上的文档格式统一，使分散的Internet资源连接为一个逻辑整体。HTML文本是由HTML命令组成的描述性文本，HTML命令可以说明文字，图形、动画、声音、表格、链接等。

css

CSS层叠样式表(英文全称：Cascading Style Sheets)是一种用来表现HTML（标准通用标记语言的一个应用）或XML（标准通用标记语言的一个子集）等文件样式的计算机语言。CSS不仅可以静态地修饰网页，还可以配合各种脚本语言动态地对网页各元素进行格式化。

javascript

JavaScript（简称“JS”） 是一种具有函数优先的轻量级， 解释型或即时编译型的编程语言。虽然它是作为开发Web 页面的脚本语 言而出名的，但是它也被用到了很多非浏览器环境中， JavaScript 基于原型编程、多范式的动态脚本语言， 并且支持面向对象、命令式和声明式 （如函数式编程）风格。

课程总结

这学期的web设计学习，让我主要是对js部分的掌握更牢了一点、对css的理解更为深刻。但是这只是静态页面，还不足以说是前端的全部工作。通过本学期的学习和期末设计，让我对前端有了初步的了解。本次期末设计个人觉得不是特别满意，因为，没有提前做好充分的准备，其实在开课前都差不多过了一遍本门课程的知识了，但是没有深入去学习，所以期末设计要用时还要去现学。就缩短了写项目的实际时间。同时，前端这部分的基础知识，html+css+js的内容个人感觉学的还是比较踏实。
